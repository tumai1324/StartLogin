StartLogin 插件版本更新日志
================================

V2.2.0 (动态对话框优化)
------------------------
- 新增：使用同一个对话框实例动态更新内容，无需关闭再打开
- 新增：DialogSession 状态管理，保存对话框类型、标题、内容、输入等状态
- 新增：updateDialogWithError() 方法，错误时直接更新对话框内容
- 新增：switchDialogType() 方法，切换对话框类型时保持流畅体验
- 新增：getInputs() 方法，获取玩家在对话框中的输入内容
- 优化：登录/注册流程使用动态对话框更新，减少闪烁
- 优化：密码修改流程使用动态对话框更新
- 修复：UILock 冷却阻塞问题，改用 forceUnlock()
- 修复：密码为空未验证问题

V2.2 (全功能版本)
------------------
- 新增：密码强度检测（大写字母、小写字母、数字、特殊字符）
- 新增：IP登录限制，同一IP最多注册指定数量账号
- 新增：登录尝试记录，记录每次登录的时间、IP、成功/失败状态
- 新增：异地登录提醒，检测异常IP登录并提示玩家
- 新增：强制修改密码，管理员可强制玩家修改密码
- 新增：密码找回功能，通过安全问题找回密码
- 新增：登录欢迎语，可自定义欢迎消息
- 新增：首次登录保护，新玩家注册后获得短暂无敌效果
- 新增：数据库自动备份，定时备份 SQLite 数据库
- 新增：玩家账号信息查询指令 /sl info <玩家名>
- 新增：在线玩家统计指令 /sl stats
- 新增：未登录玩家自动踢出，超时未登录的玩家会被踢出
- 新增：密码过期策略，密码超过指定天数后强制修改
- 新增：最小账号时长限制，防止小号刷号

V2.1.5 (Bug修复)
-----------------
- 修复：会话缓存丢失问题（V2.1.3误将sessionCache.remove加到removeOnlineSession中）
- 优化：过期缓存在isSessionCached()检查时自动清理

V2.1.4 (Bug修复)
-----------------
- 修复：内存泄漏问题，玩家退出时清理相关数据
- 修复：UILock 冷却阻塞后续操作问题

V2.1.3 (Bug修复)
-----------------
- 修复：openMessageDialog NullPointerException（DialogType.confirmation()需要两个非空按钮）
- 修复：Folia 调度器兼容性问题，使用线程池替代Bukkit.getScheduler()

V2.1.2 (功能完善)
-----------------
- 新增：调试日志开关配置
- 新增：密码黑名单功能
- 优化：消息显示从聊天框改为对话框
- 优化：美化指令帮助信息，加入颜色和图标

V2.1.1 (UI优化)
----------------
- 新增：对话框日志功能
- 优化：指令帮助信息美化

V2.1 (基础功能)
----------------
- 新增：所有玩家首次进入需注册+同意规则
- 新增：未登录玩家限制（不能移动、破坏、受伤等）
- 新增：会话缓存功能，规定时间内免密码登录
- 新增：支持 Paper / Folia 1.21 ~ 26.1
- 新增：SQLite 嵌入式数据库存储

V2.0 (重构版本)

- 新增：线程池管理，支持 Folia 环境

V1.0 (初始版本)
